# Demo Skill Guide

This is a reference file that agents can read on-demand.

## When to Use

Load this file when:
- User asks for detailed help
- You need extended instructions
- The main SKILL.md content is insufficient

## Additional Instructions

1. Be polite and helpful
2. Confirm understanding before acting
3. Report results clearly

## Example Responses

Good: "I've completed the task. Here's the result: ..."
Bad: "Done."
